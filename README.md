<h3 align="center">Blog de Gravity Falls</h3>

- PASO 1 ➡️ Descargará todo lo necesario para poder ejecutar el proyecto **npm install**

- Aquí se encuentra el back-end del blog [LAB6-WEB](https://github.com/nancygmm/LAB6-WEB.git)

- PASO 2 ➡️ Se debe de poner a correr el back-end del blog **docker compose up --build**

- PASO 3 ➡️ Se debe poner a correr el front-end del blog **npm run dev**
